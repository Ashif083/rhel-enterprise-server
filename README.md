# rhel-enterprise-server
This project demonstrates the design and implementation of a production-style Linux server using Red Hat Enterprise Linux. <br>
The goal was to simulate a real-world enterprise environment including:<br>
-User management<br>
-Storage management (LVM)<br>
![LVM](screenshots/lvm.png) <br>
-Web server deployment<br>
![Web](screenshots/apache_host.png) <br>
-Security hardening<br>
![Permissions](screenshots/permissions.png) <br>
-Automated backup<br>
-System monitoring<br>
